# CARLoS-Agents
CARLoS simulator with multiple agent implementations

step 1: clone the repo or download the zip file. if zip file extract the file locally. \n
step 2: open command prompt and cd to the project folder carlos_agents. \n
step 3: create a virtual environment by running command "python -m venv venv"\n
step 4: activate the venv by running "venv\Scripts\activate" \n
step 5: "pip install -r requirements.txt" \n
step 6: in the src folder > layouts> find the train_path.txt file and copy its file path. \n
step 7: in rlagent > carlos_gym_env.py > layout file_path variable update with the copied file path. \n
step 8: in command prompt run "python -m rlagent.[filename]" \n
